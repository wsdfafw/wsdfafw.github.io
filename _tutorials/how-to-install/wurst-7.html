This installation tutorial is for Minecraft {{ site.latest_mcversion }} - 1.14.2 with Fabric Loader, as well as for snapshot versions. If you want to install Wurst for Minecraft 1.12.2 or older versions, please follow the Wurst 6 installation guide instead. If you want to install ForgeWurst, please follow the ForgeWurst installation guide instead.

{%- include lang-select.html -%}

Step 1: Install Fabric (Loader)

  1. Download and run the Fabric installer.

    Troubleshooting

    {%- include troubleshooting/fabric-installer-antivirus.html -%} {%- include troubleshooting/fabric-installer-nothing-happens.html -%} {%- include troubleshooting/fabric-installer-opens-winrar.html -%} {%- include troubleshooting/fabric-installer-no-java-installed.html -%}

  2. In the Fabric installer, select your Minecraft version and press "Install".

    • If you want to use snapshots, make sure that "Show snapshots" is checked first.
    • Leave "Select Install Location" at the default value. Changing this can sometimes cause problems.
    • Leave "Create profile" checked. That way you don't have to make your own profile.

Step 2: Install Fabric API

  1. Go to the Fabric API page and select your Minecraft version at the top right.

    Then download the latest (top-most) version of Fabric API for that Minecraft version.

    Troubleshooting

    {%- include troubleshooting/fabric-api-cant-find-download-button.html -%} {%- include troubleshooting/fabric-api-not-commonly-downloaded.html -%}

  2. Place the Fabric API in your .minecraft/mods folder.

    Troubleshooting

    {%- include troubleshooting/fabric-api-cant-find-minecraft-folder.html -%} {%- include troubleshooting/fabric-api-there-is-no-mods-folder.html -%} {%- include troubleshooting/fabric-api-doesnt-have-java-icon.html -%}

Step 3: Install Wurst

  1. Download the Wurst Client, if you haven't already. Make sure to get the latest version that is compatible with your Minecraft version.

    Troubleshooting

    {%- include troubleshooting/wurst-not-commonly-downloaded.html -%}

  2. Place the Wurst Client in your .minecraft/mods folder.

    Troubleshooting

    {%- include troubleshooting/wurst-cant-find-minecraft-folder.html -%} {%- include troubleshooting/wurst-there-is-no-mods-folder.html -%} {%- include troubleshooting/wurst-doesnt-have-java-icon.html -%}

Step 4: Launch The Game

  1. In the Minecraft Launcher, open the Installations tab and then launch your new fabric-loader-... installation.

    Troubleshooting

    {%- include troubleshooting/launch-player-safety-warning.html -%} {%- include troubleshooting/launch-cant-see-loader-installation.html -%} {%- include troubleshooting/create-profile-automatically.html -%} {%- include troubleshooting/rename-loader-installation.html -%} {%- include troubleshooting/create-profile-manually.html -%}

  2. That's it! You are now using Wurst.
