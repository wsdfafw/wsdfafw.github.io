This installation tutorial is only for Wurst 6 (Minecraft versions 1.12.2 - 1.7.2), using the MultiMC launcher or one of its forks (PolyMC, Prism Launcher, etc). If you want to install other versions of the Wurst Client or use different launchers, please follow the appropriate guide below:

  • Wurst 6 using the official Minecraft launcher
  • Wurst 7 for Minecraft 1.14.2 and newer versions
  • ForgeWurst for Minecraft 1.12.2 - 1.10.2

Step 1: Create a New Instance in MultiMC

  1. Open MultiMC and click the "Add Instance" button.

  2. From the "Vanilla" tab, select the correct Minecraft version for installing your selected version of Wurst. If you don't know which Minecraft version to use, please refer to the table below.

    {%- include how-to-install/wurst-6/installer-compatibility-table.html -%}

    Troubleshooting

    {%- include troubleshooting/wurst-6-worried-server-compatability.html -%}

  3. Give your instance a name, optionally assign it to a group, and click the "OK" button.

Step 2: Extract the Files to a Temporary Folder

  1. Download the Wurst Client, if you haven't already. Make sure to get the latest version that is compatible with your Minecraft version.

    Troubleshooting

    {%- include troubleshooting/wurst-6-installer-jar-or-zip.html -%} {%- include troubleshooting/wurst-not-commonly-downloaded.html -%}

  2. Open the Wurst 6 installer and click the "Extract" button.

    Troubleshooting

    {%- include troubleshooting/wurst-6-installer-wont-open.html -%} {%- include troubleshooting/wurst-6-installer-cant-extract.html -%}

  3. Select a temporary folder to extract the files to. Be sure to remember the location for later.

Step 3: Edit the Instance to Use Wurst 6

  1. Select your MultiMC instance and click the "Edit Instance" button.

  2. Under the "Version" tab, click the "Replace Minecraft.jar" button.

  3. Navigate to the temporary folder where you extracted the Wurst 6 files earlier and select the .jar file inside.

Step 4: Launch the Game

  1. Select your MultiMC instance and click the "Launch" button.

  2. That's it. Enjoy using Wurst 6 in MultiMC!
