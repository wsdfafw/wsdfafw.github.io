这是我们为使 Wurst 成为世界上最安全的 Minecraft 作弊客户端所做的工作：

开源软件

Wurst 客户端是开源软件，完整的 Wurst 7 源代码 在 GitHub 上对所有人开放。我们没有任何可隐瞒的东西！

无混淆

当我们说我们没有什么可隐藏的时候，我们是认真的！与我们的竞争对手不同，我们不使用任何混淆技术。

这意味着您始终可以对 Wurst 进行反编译，查看其操作，即使您没有源代码也可以做到。此外，这还意味着您始终可以验证 Wurst 实际上是基于我们提供给您的源代码，而且很容易发现伪造。

加密账号管理器

Wurst 的账号管理器以加密格式保存您的账号信息，而大多数其他客户端将其以明文形式存储。这使得 Wurst 客户端更具抵抗力，可以抵御病毒和试图窃取您的 Minecraft 账号的人。

Scanning for Viruses

Every change to the Wurst Client's code is automatically uploaded to VirusTotal and scanned with 60+ different antivirus programs, including Kaspersky, BitDefender, MalwareBytes, Windows Defender and so many more. In the rare event that one of them gives a false positive, we act immediately to resolve the issue.
