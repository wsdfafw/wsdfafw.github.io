This version of the Wurst Hacked Client is available for Minecraft 1.16.5, 1.16.4 and 1.16.3, as well as snapshots 21w20a and 21w19a.

## Changelog

- Fixed a crash when running the [.potion](https://wurst.wiki/cmd/potion) command with an invalid potion effect.

- Searching for "AirJump" in Navigator will now find [Jetpack](https://wurst.wiki/jetpack).

- Fixed [Nuker](https://wurst.wiki/nuker) not rendering properly when extremely far from spawn.

- Fixed [NukerLegit](https://wurst.wiki/nukerlegit) not rendering properly when extremely far from spawn.

- Fixed [BuildRandom](https://wurst.wiki/buildrandom) not rendering properly when extremely far from spawn.

- Fixed [Killaura](https://wurst.wiki/killaura) not rendering properly when extremely far from spawn.

- Fixed [KillauraLegit](https://wurst.wiki/killauralegit) not rendering properly when extremely far from spawn.

- Fixed [MobSpawnESP](https://wurst.wiki/mobspawnesp) crashing the game in Minecraft 21w19a and later versions.

- Fixed [Tunneller](https://wurst.wiki/tunneller) not rendering properly in Minecraft 21w19a and later versions.
