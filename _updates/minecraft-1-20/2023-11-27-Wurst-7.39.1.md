## Changelog

- 修复了 BitDefender 和一些较为不知名的杀毒软件的误报问题。([之前](https://www.virustotal.com/gui/file/60456d19ba513e15599d5ccb22eaf1409b76af2fe4372c3f2c180c9e0bc49583) / [之后](https://www.virustotal.com/gui/file/41e6c72b588d5befbcba4ddf7a2e8a9e88c3b68a6d720436e4fadc54f86c10dc))

似乎杀毒软件的人工智能主宰突然决定不喜欢 AutoBuild 中的某一部分，而这部分代码已经存在多年。因此，许多 Wurst 版本现在被错误地标记为特洛伊木马。

目前，我发布了一个新版本，以 [稍微不同的方式](https://github.com/Wurst-Imperium/Wurst7/commit/97d1225f72feb3e2f945884edd6769320b0c8c19) 执行 AutoBuild 中的这一部分，以便 BitDefender 停止对其提出异议。<del>但为了更长期的解决方案，让我们一起 [向 BitDefender 报告此问题](https://www.bitdefender.com/consumer/support/answer/29358/)，以便他们也能在他们的端解决误报问题。</del> <b>更新：</b> BitDefender 似乎已经移除了误报。

另请参阅：[Wurst 7.39 更新日志](/updates/wurst-7-39/)

## Keybinds

All keybinds can be changed in-game. Go to Wurst&nbsp;Options > Keybinds or type [`.help binds`](https://wurst.wiki/cmd/binds) in the chat.

<details>
  <summary>默认按键绑定：（点击展开）</summary>
  <ul>
    <li>B -> 快速放置；快速破坏</li>
    <li>C -> 全亮</li>
    <li>G -> 飞行</li>
    <li>Ö -> 速度挖掘者</li>
    <li>H -> /home</li>
    <li>J -> 耶稣（Jesus）</li>
    <li>K -> 多重光环（MultiAura）</li>
    <li>N -> 毁灭者（Nuker）</li>

    <li>右CTRL -> 点击界面（ClickGUI）</li>
    <li>右Shift -> 导航器（Navigator）</li>
    <li>U -> 自由视角（Freecam）</li>
    <li>X -> X射线</li>
    <li>Z -> 潜行</li>
  </ul>
</details>
